Week 9: Object-Oriented Math Library in C++
A C++ program demonstrating OOP concepts using a hierarchy of numeric classes.

Concepts Covered:

Inheritance (Number → Adv_Number → Custom_Num)
Function Overloading (add, subtract, multiply, divide with 1 or 2 arguments)
Operator Overloading (+, -, *, /, ++, >>, <<)
Encapsulation (protected data member x accessed via methods)
Friend Functions (for stream operators >> and <<)
Files:

mathlib.h — Base and derived class definitions.
main.cpp — Custom_Num class and driver code.
